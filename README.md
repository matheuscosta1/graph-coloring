## Para executar
- Instale Stack: 
https://docs.haskellstack.org/en/stable/install_and_upgrade/ , ou, para sistemas *nix simplesmente use:

```bash
curl -sSL https://get.haskellstack.org/ | sh
```

- Na raiz do projeto, rode:

```bash
stack run
```
